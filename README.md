# PingPongChallenge
Coding Challenge: Ping Pong 
1. Clone this repository
2. `cd ping-pong-challenge`
3. Run `npm install`
4. Run `node server`
5. In a second command window, `cd ping-pong-challenge`, Run `npm start`
